# LMS Pendidikan Komputer

LMS Pendidikan Komputer adalah platform pembelajaran digital yang dibuat untuk membantu proses perkuliahan menjadi lebih terstruktur, mudah diakses, dan mendukung pembelajaran mandiri mahasiswa. Website ini dikembangkan sebagai media belajar berbasis WordPress dengan tampilan sederhana, modern, dan mudah digunakan.

## Filosofi Pengembangan

LMS ini dibuat dengan filosofi bahwa pembelajaran digital seharusnya tidak hanya menjadi tempat menyimpan materi, tetapi juga menjadi ruang belajar yang membantu mahasiswa memahami alur perkuliahan secara lebih jelas. Setiap bagian dalam LMS dirancang agar mahasiswa dapat menemukan materi, tugas, dan informasi mata kuliah dengan mudah tanpa harus merasa bingung saat menggunakan website.

Pengembangan LMS ini juga berangkat dari kebutuhan mahasiswa untuk memiliki media pembelajaran yang fleksibel. Dengan adanya LMS, proses belajar tidak hanya bergantung pada ruang kelas, tetapi dapat dilakukan kapan saja dan di mana saja. Website ini diharapkan dapat menjadi jembatan antara dosen, materi pembelajaran, dan mahasiswa dalam kegiatan perkuliahan.

## Tujuan Website

* Menyediakan media pembelajaran digital untuk mahasiswa.
* Memudahkan mahasiswa mengakses daftar mata kuliah.
* Menyusun materi pembelajaran berdasarkan mata kuliah dan pertemuan.
* Mendukung kegiatan belajar mandiri secara online.
* Membuat tampilan LMS yang sederhana, rapi, dan mudah digunakan.

## Fitur Utama

* Landing page sebagai halaman awal website.
* Halaman home sebagai pusat informasi LMS.
* Daftar mata kuliah.
* Halaman detail mata kuliah.
* Materi pembelajaran setiap mata kuliah.
* Tampilan responsif untuk desktop dan perangkat mobile.
* Navigasi sederhana agar pengguna mudah berpindah halaman.

## Teknologi yang Digunakan

* WordPress sebagai platform utama.
* LearnPress sebagai plugin LMS.
* Astra Theme sebagai tema dasar tampilan.
* HTML untuk struktur halaman.
* CSS untuk desain dan tampilan website.
* PHP sebagai bahasa utama WordPress.
* MySQL sebagai database WordPress.

## Struktur Halaman

* Landing Page
  Halaman awal yang memperkenalkan website sebelum pengguna masuk ke beranda.

* Home
  Halaman utama yang berisi penjelasan LMS, fitur pembelajaran, dan akses menuju mata kuliah.

* Mata Kuliah
  Halaman yang menampilkan daftar mata kuliah yang tersedia di dalam LMS.

* Detail Mata Kuliah
  Halaman yang berisi informasi dan materi pembelajaran dari setiap mata kuliah.

## Konsep Desain

Desain LMS ini menggunakan konsep sederhana, bersih, dan ramah pengguna. Warna biru dipilih untuk memberikan kesan profesional, tenang, dan mendukung suasana belajar. Tampilan dibuat tidak terlalu ramai agar mahasiswa dapat fokus pada konten pembelajaran.

Prinsip desain yang digunakan:

* Tampilan sederhana dan tidak membingungkan.
* Navigasi mudah dipahami.
* Warna konsisten pada setiap halaman.
* Konten disusun secara rapi.
* Fokus pada kenyamanan pengguna.

## Manfaat

LMS ini memberikan manfaat bagi mahasiswa dan pengelola pembelajaran, antara lain:

* Mahasiswa lebih mudah mengakses materi.
* Materi perkuliahan tersusun berdasarkan mata kuliah.
* Proses pembelajaran menjadi lebih fleksibel.
* Website dapat digunakan sebagai media pendukung perkuliahan.
* Tampilan LMS dapat dikembangkan sesuai kebutuhan pembelajaran.

## Cara Menjalankan Website

1. Siapkan server lokal seperti XAMPP, Laragon, atau LocalWP.
2. Letakkan folder website pada direktori server lokal.
3. Import database WordPress ke phpMyAdmin.
4. Sesuaikan konfigurasi database pada file `wp-config.php`.
5. Jalankan server lokal.
6. Buka website melalui browser.

Contoh alamat lokal:

```text
http://localhost/wordpress/
```
